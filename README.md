# MSc Dissertation: TLS Malicious Traffic Detection

Detection of malicious TLS-encrypted network traffic using flow-level
statistical features, without payload decryption.

Module: 7COM1070
University of Hertfordshire, MSc Cyber Security, 2025/26

## Structure
- data/         Raw PCAPs and extracted feature CSVs (gitignored)
- notebooks/    Jupyter notebooks for exploration and analysis
- src/          Reusable Python modules
- results/      Model outputs, SHAP plots, evaluation metrics
- report/       Dissertation drafts
- diary/        Weekly progress reports
- scripts/      Shell scripts for data download and processing
